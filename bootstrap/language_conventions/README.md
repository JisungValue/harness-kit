# Language Conventions Bootstrap

이 디렉터리는 언어별 coding convention 초안을 제공한다.

## 사용 방법

- 단일 언어 프로젝트면 필요한 언어 템플릿을 `docs/project/standards/coding_conventions_project.md`에 복사 또는 병합한다.
- 다중 언어 프로젝트면 공통 규칙은 `coding_conventions_project.md`에 두고, 언어별 상세 규칙은 별도 문서로 복사한 뒤 프로젝트 문서에서 참조한다.
- 사용하지 않는 언어 템플릿까지 모두 프로젝트에 복사하지 않는다.

## 현재 언어

- `python_coding_conventions_template.md`
- `java_coding_conventions_template.md`
- `kotlin_coding_conventions_template.md`
